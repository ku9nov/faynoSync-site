# JavaScript SDK

Coming soon.

The JavaScript / TypeScript SDK will provide the same goals as the [Go SDK](/docs/sdk/go/sdk): typed update checks, optional edge-first delivery, and client-side optimizations without requiring changes to the core faynoSync API.

For now, use the [Go SDK](/docs/sdk/go/sdk) or call `/checkVersion` on your faynoSync API directly from your client.
