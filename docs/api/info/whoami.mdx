# Check current user information

**Endpoint:**

`GET /whoami`

**Headers:**

`Authorization`: Bearer JWT token.

**Request:**

```bash
curl -X GET http://localhost:9000/whoami \
-H "Authorization: Bearer <jwt_token>"
```
