# Get All Applications (deprecated)

Retrieve a list of all applications.

**Endpoint:**

`GET /`

**Headers:**

`Authorization`: Bearer JWT token.

**Request:**

```bash
curl -X GET http://localhost:9000/ -H "Authorization: Bearer <jwt_token>"
```
