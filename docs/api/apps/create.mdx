# Create Application

Creates a new application.

Optional with `description`, `logo`, `private`. 

**Endpoint:**

`POST /app/create`

**Headers:**

`Authorization`: Bearer JWT token.

**Body (multipart/form-data):**

- `app`: The name of the application to be created.

- `file`: Logo of the app.

- `description`: App description.

- `private`: Lock app. (If selected, the app will be stored in a private bucket. Can't be changed.)

**Request:**

```bash
curl --location 'http://localhost:9000/app/create' \
--header 'Authorization: Bearer <jwt_token>' \
--form 'data="{\"app\":\"appName\"}"'
```

**Response:**

```json
{
   "createAppResult.Created":"641459ffb8760d74164e7e3c"
}
```
