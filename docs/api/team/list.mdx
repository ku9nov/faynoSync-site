# List Users

Retrieve a list of all your team users.

**Endpoint:**

`GET /users/list`

**Headers:**

`Authorization`: Bearer JWT token.

**Request:**

```bash
curl -X GET http://localhost:9000/users/list \
-H "Authorization: Bearer <jwt_token>"
```
